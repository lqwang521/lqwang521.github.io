《扫除道》

发表于 2019-03-17 | 分类于 阅读 | 阅读次数: |

阅 读 收 获

  1. 提供变得谦虚的方法
  2. 介绍如何提升觉察力
  3. 培养感动之心
  4. 学会感恩
  5. 提供磨练心性的途径

金 句 精 选

  1. 特别是身处逆境的时候，把周围打扫干净，会给自己一种得救的感觉。

2.再没有比被期待、被依靠更能让人感到愉悦的事了。

3.无论做什么，别人最最不愿干的事，首先自己要承担起来，这是调动他人工作积极性的前提。　

作 者 简 介

[日] 键山秀三郎

生于1933年，“扫除道”创始人。他在1961年创立ROYAL公司（后更名为YELLOW HAT），1998年出任该公司顾问，现任“日本美好协会”顾问。著有《凡事彻底》《键山秀三郎语录》《从扫除中学到的人生法则》等。

精 华 解 读

以下内容为《扫除道》一书精华解读，供广大书友们学习参考，欢迎分享，未经允许不可用作商业用途。

目 录

一、扫除的变迁
二、扫除的基本要求和范围
三、扫除道的五大好处
四、扫除让什么发生了改变

一、扫除的三个基本要求

  1. 备齐扫除工具
  2. 确定扫除工具的存放处
  3. 扫除过程中不怕动脑

二、扫除的五大好处

  1. 使人谦虚
  2. 让人成为有心人
  3. 孕育感动
  4. 萌生感恩
  5. 磨炼心性

三、做到”凡事彻底的“的三个必要条件

  1. 任何事情做到无微不至
  2. 毫不犹豫地坚持自己的主义和行动
  3. 充分发挥一切资源的作用

总结

1. 清扫一个地方，就赋予了其生命，扫除者将受益于它

2. 从未见过乐于扫除的人是个傲慢者。而那些坚持扫除的人，都变成了谦逊的人

3. 感恩细小事物的人，会人为地缩小痛苦和烦恼。反之，不能感恩细小事物的人，则会人为地放大痛苦和烦恼

4. 俗话说：知道的是理，行出来的才是道
